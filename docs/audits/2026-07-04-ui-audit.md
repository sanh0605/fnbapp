# UI Audit — Web Interface Guidelines Compliance

Date: 2026-07-04
Scope: `app/admin/**` + `components/**` (104 tsx files)
Source: [Vercel Web Interface Guidelines](https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md)
Method: Grep mechanical scan + per-file context read on shared components + pilot files

## Executive Summary

15 systemic issues. 6 are CRITICAL — they affect every page in the app because they live in shared components (`FormModal`, `LoadingButton`, `DeleteConfirmModal`) or are completely absent (`focus-visible`, `aria-live`, `Intl.*`).

Top 3 to fix first:

1. **FormModal a11y** — every modal inherits broken behavior (no Escape, no focus trap, no role).
2. **Vietnamese diacritics in shared components** — DeleteConfirmModal + LoadingButton ship ASCII Vietnamese ("Huy", "Xoa", "Dang xu ly...") shown on every delete/submit.
3. **`focus-visible:ring-*` everywhere** — keyboard users cannot see focus target system-wide (0 occurrences vs 85 `transition-all`).
